幫助： Profile

描述

Profile 模組用於管理自定義的用戶資料欄位。

安裝/卸載

無需特別措施，請按照標準安裝過程進行 – 將 /profile 資料夾解壓至 ../modules 目錄中。通過管理界面安裝模組 -> 系統模組 -> 模組。

有關模組安裝的詳細說明，請參閱 XOOPS 操作手冊

操作說明

要設置此模組，您需要進行以下操作：

  • 配置您的模組偏好設置（請參見「偏好設置」），如果您打算使用 Profile 區塊，則選擇性地配置區塊（請參見「區塊」）。
  • 編輯現有的類別或新增類別。
  • 編輯現有的欄位或新增欄位。在這裡，您可以指定哪些欄位會顯示在哪些類別中，並且這些欄位是否會在用戶註冊時顯示。
  • 定義註冊步驟的順序。
  • 最後，您可以設置每個欄位的權限 – 哪些欄位是可編輯的，哪些欄位是可搜尋的。

防止垃圾郵件措施

為了減少垃圾註冊，請執行以下操作：

  • 進入 Admin 中的 Protector 模組，進入偏好設置，然後在底部選擇此選項： "停止論壇垃圾郵件" 檢查 POST 數據是否與 www.stopforumspam.com 數據庫中的垃圾郵件發送者匹配。
    將其設置為 "封鎖 IP（無限制）"

  • 在 /class/captcha/config.php 中，確保模式設置為 "text"：

    return $config = array(
    'disabled' => false, // 禁用 CAPTCHA
    'mode' => 'text', // 默認模式，您可以選擇 'text'，'image'，'recaptcha'（需要 API 密鑰）
    'name' => 'xoopscaptcha', // captcha 名稱
    'skipmember' => true, // 跳過會員 CAPTCHA 驗證
    'maxattempts' => 10, // 每個會話的最大嘗試次數
    );

  • 在 Profile 模組中，進入 Admin，並在位於以下位置的基本步驟中：
    /modules/profile/admin/step.php?id=1
    將 "停止後保存" 設置為 "否"


  • 在 Profile 偏好設置中，將 "在第二步註冊後使用 Captcha" 選項設置為 "是"（默認設置）
