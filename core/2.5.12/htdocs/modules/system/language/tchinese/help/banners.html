說明：橫幅廣告

描述

橫幅廣告是顯示在頁面頂部區域的廣告。它們可以是圖片或 Flash 影片，例如預設安裝中包含的 XOOPS 橫幅廣告，其特色是大幅旋轉的動畫。XOOPS 內建了一個用於管理橫幅廣告和建立廣告客戶帳戶的系統，該系統位於「系統模組 -> 橫幅廣告」。您可以在 XOOPS 網站上設置多個橫幅廣告 - 每次載入新頁面時，顯示區域會輪流展示每個啟用的橫幅廣告。

每次橫幅廣告向訪客展示時，這將計為一次「曝光」，這是橫幅廣告的常見交易單位。客戶通常會支付一定數量的曝光費用 - 也就是讓他們的廣告顯示特定次數。

請注意：如果您從事基於曝光次數銷售橫幅廣告的業務，有一件重要的事情需要做：前往「系統模組 -> 偏好設定 -> 一般設定」，並填寫「您的 IP 地址」欄位（如果您有一個固定的 IP 地址）。這將防止您對網站的訪問被計入曝光次數。如果不這樣做，可能會被認為有些不誠實（然而，請注意，許多管理員沒有固定 IP - 例如使用撥號數據機的使用者）。
