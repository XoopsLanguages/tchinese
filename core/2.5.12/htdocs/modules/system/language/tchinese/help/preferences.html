幫助：偏好設定

描述

偏好設定面板是您配置網站全域設定的地方。花時間進行設定是值得的，因為有些設置會對網站的效能、安全性和使用便捷性產生重要影響。

- 清理快取資料夾
- 清空會話表
- 執行資料庫表的維護（分析、檢查、優化和修復）

您還可以對整個資料庫或個別資料表執行 "備份" 操作。

一般設定

這些是最重要且最常被訪問的設定。您可以設置網站名稱和口號。您還可以選擇預設主題和管理界面。如果您需要暫時停止網站，您可以選擇將其關閉給所有用戶。您也可以設置模組的快取時間。

Meta 標籤和頁腳

Meta 標籤提供有關您網站的信息——它們描述網站的性質。它們的主要用途是幫助搜尋引擎及其爬蟲正確地索引您的網站，即幫助正確的讀者找到它。您提供的信息會被包含在每個網頁的頭部。它應該代表整個網站的內容——Meta 信息在每個頁面上都是一樣的，目前無法為不同的模組或頁面自定義。

請注意，不同的搜尋引擎對於網站的索引策略有所不同。他們可能會對 Meta 標籤給予不同的權重，甚至完全忽略它們。謹慎使用 Meta 信息可能有助於您的網站在搜尋引擎中獲得更好的排名，並且了解您主要參照來源的索引策略（查看網站日誌）也很有幫助。這將幫助您優化 Meta 信息或網站設計和內容的其他方面，從而提高排名。

嘗試準確且誠實地提供 Meta 信息。不要試圖“操控”搜尋引擎來獲得更高排名——它們可能會識別到這種行為，並可能將您的網站降級或從索引中黑名單處理。

Meta 版權 和 頁腳 欄位都支持兩個替代標籤以簡化維護。
 - {X_SITEURL} 會被替換為您網站主頁的完整 URL。
 - {X_YEAR} 會被替換為當前年份。這樣可以避免每年更新版權日期。

詞彙過濾選項

XOOPS 可以自動過濾網站訪客發表的令人不快的詞彙，並將其替換為您選擇的通用替代詞。過濾功能會消耗一些系統資源，可能會稍微降低網站效能。默認情況下它是禁用的。

若要在您的網站上啟用過濾，選擇‘啟用’單選框。您需要在‘要過濾的詞彙’框中輸入不需要的詞彙，並用管道符號分隔它們。此功能不區分大小寫。在‘用來替換不良詞彙的詞語’框中輸入適當的替換詞。默認是 #OOPS#。

搜尋選項

搜尋功能是定位門戶網站中信息的最重要工具之一。XOOPS 搜尋將結果按模組分類——記錄會被標識為新聞項目、下載項目、鏈接或其他類型，根據您安裝的模組。

預設情況下會啟用對您網站上所有資源的全域搜尋。如果您禁用全域搜尋，則此功能將不再可用，搜尋區塊將變為無效。

搜尋的最小關鍵字長度預設為五個字符。任何更短的詞彙將被忽略。這有助於控制搜尋的精確度，雖然對像“網站地圖”這類詞彙沒有太大用途。較短的關鍵字通常更常見，因此返回的記錄數量較多，其中許多可能是不相關的。較長的關鍵字通常檢索到較少的記錄，但精確度較高。

搜尋在計算上相對昂貴，因此排除短語將減少伺服器負擔。然而，請記住，有時能夠搜尋短詞還是很有用的。

電子郵件設置

控制面板的此部分用於配置您的系統電子郵件。建議您將電子郵件發送方法保留在默認設置 “PHP (mail)” 上。除非您使用替代的電子郵件發送選項，否則不需要填寫 Sendmail 或 SMTP 欄位。

認證選項

您可以將其保持為 Xoops 資料庫，並忽略那裡的其他選項。

目前已經為認證添加了 LDAP 或 Microsoft Active Directory 伺服器作為 Xoops 用戶的替代選擇。原因是許多公司已經擁有 ADS 或 LDAP 用戶資料庫，他們可以使用該資料庫登錄 XOOPS，而不需要註冊另一個帳戶。

系統模組設置

在這裡，您可以啟用/停用每個系統區塊，並指定全域預設設置，例如每頁顯示的頭像數量或圖片數量。您還可以為區塊、評論以及所有模組選擇預設編輯器。
