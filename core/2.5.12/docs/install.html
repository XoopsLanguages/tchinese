安裝 XOOPS

首次安裝

前言：

XOOPS 是一個開源的 物件導向網頁發布系統，使用 PHP 編寫。它是開發小型到大型動態社群網站、企業內部門戶、企業門戶、網誌等的理想工具。

XOOPS 採用 GNU 通用公共許可證 (GPL) 版本 2 或更高版本發佈，並且可以自由使用和修改。 只要遵守 GPL 的分發條款，就可以自由再分發。

需求

  • WWW 伺服器 (Apache、NGINX、IIS 等)
  • PHP 5.6.0 或更高版本，推薦使用 7.4+ 版本
  • MySQL 5.6 或更高版本，推薦使用 MySQL 8.0+ 版本

安裝前準備

  1. 正確設定 WWW 伺服器、PHP 和資料庫伺服器。
  2. 為你的 XOOPS 網站準備一個資料庫。
  3. 準備用戶帳號並賦予該用戶對資料庫的訪問權限。
  4. 使以下目錄和文件可寫：%s
  5. 為了安全考慮，強烈建議將以下兩個目錄移出 文件根目錄，並更改文件夾名稱：%s
  6. 創建（如果尚未存在）並使以下目錄可寫：%s
  7. 開啟瀏覽器的 cookies 和 JavaScript。

本地安裝

如果你正在運行開發或測試的本地環境，請確保滿足之前的需求。一旦完成，將 htdocs 目錄（來自 XOOPS 發行檔案）的內容複製到你的網頁環境的根目錄。文件複製完成後，你可以通過瀏覽器指向 "http://yoursite.com" 開始安裝過程。

在託管平台上安裝

如果你在託管環境中運行，請將 XOOPS 檔案解壓到本地或伺服器上（如果你有 shell（例如 SSH）訪問權限）。完成後，請確保將所有 XOOPS 檔案從 htdocs 目錄移動或複製到你的根網頁目錄（你的提供商通常會提供這個位置及其指示）。檔案複製完成後，你可以通過瀏覽器指向 "http://yoursite.com" 開始安裝過程。

繼續安裝

只需按照安裝程序中的逐步說明操作即可。
